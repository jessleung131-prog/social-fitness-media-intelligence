Attribution window mismatch
Reddit fixed at 30d_click vs TikTok/Meta 7d_click. Reddit installs appear inflated relative to spend. Never SUM installs across platforms — use AppsFlyer MMP as single source of truth.
Instagram watch time in ms
Instagram Graph API returns avg_watch_time_ms in milliseconds. Transformer divides by 1000 before CDM write. Forgetting this gives Instagram an apparent 1000× advantage in watch time vs TikTok.
SKAN rows — no user join
19.1% of iOS installs are is_skan = TRUE (ATT-denied). Empty customer_user_id. Cannot be joined to cdm_funnel_events. Count in install totals but exclude from user-level funnel analysis.
Meta ROAS as nested list
Meta purchase_roas is not a float — it's a list of action dicts: [{"action_type":"offsite_conversion...","value":"1.23"}]. Transformer extracts [0]["value"] and casts to float. Other parsers fail silently.

Five CDM tables. Schema differences absorbed at transform time — downstream SQL never sees platform-specific field names.

cdm_installs
AppsFlyer
57,999 rows · One row per install · Full funnel flags
FieldTypeDescription
appsflyer_idSTRINGMMP device ID. Primary join key. Pseudonymous.
customer_user_idSTRINGApp user ID. Empty until registration. NULL for SKAN.
install_dateDATEUTC date stripped from install_timestamp
channelSTRINGTikTok Ads / Meta Ads / Reddit Ads / Organic
att_statusSTRINGauthorized / denied / not_determined / not_applicable
is_skanBOOLTrue → SKAN only. No user-level join possible.
subscribedBOOLPaid subscription activated post-install
subscription_value_usdFLOATRevenue attributed to this install
days_to_subscribeINT41% subscribe Day 0. Median = Day 1.

FieldTypeDescription

content_typeSTRINGREELS, Standard Video, Text Post, etc.
